Release Plans
=============

This folder contains a complete, step-by-step implementation plan to publish ManiPDF as GitHub Release assets for Windows (NSIS installer) and macOS (universal DMG). Read the files in the order below.

- release-plan.md            — High-level plan and phased steps
- windows-nsis.md           — Windows build + NSIS installer details
- macos-dmg.md              — macOS universal2 .app and DMG workflow
- ci-github-actions.md      — Example GitHub Actions workflow (text only)
- pyinstaller-spec.md       — Example PyInstaller .spec templates and tips
- checklist.md              — Pre-release and post-release checklist

Follow the guidance in release-plan.md first; the remaining files contain the detailed commands, snippets and troubleshooting notes you will need when implementing the pipeline.
